# homebrew-tap

Homebrew tap for [ramus](https://github.com/1337raspberry/ramus) —  A highly opinionated, genre-first music player for Plex on macOS. 

## Install

```bash
brew install 1337raspberry/tap/ramus
```

Or:

```bash
brew tap 1337raspberry/tap
brew install --cask ramus
```

## Update

```bash
brew upgrade ramus
```
