# homebrew-tap

Homebrew tap for [ramus](https://github.com/1337raspberry/ramus) — a keyboard-driven Plex music player for macOS.

## Install

```bash
brew install 1337raspberry/tap/ramus
```

Or:

```bash
brew tap 1337raspberry/tap
brew install --cask ramus
```

## Update

```bash
brew upgrade ramus
```
